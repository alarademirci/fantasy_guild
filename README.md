## How to Run Locally
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python3 app.py

3. Open your browser at `http://127.0.0.1:3000`

IMPORTANT NOTE: `SIMULATED_NOW` in `app.py` (under CONSTANTS) controls the fictional current date/time used for participation deadline checks. It is currently set to Monday 17 August 2026, 09:00. You can change it to test different scenarios.
